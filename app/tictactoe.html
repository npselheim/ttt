Tic-Tac-Toe Demo

Programming Tic-Tac-Toe is not difficult and is generally considered a high school level exercise. My objective here is to learn the JavaScript language and to try out some best practices with respect to JavaScript design.

  • single global variable
  • keep html, css, and js separate
  • one class per file
  • unit tests with QUnit
  • documentation with JSDoc
  • use array idioms instead of for loops
  • pass JSLint
