Tic Tac Toe demo

For now, all you can do is play with yourself, or two people can play if they want to hand off the mouse. Eventually I will implement a program player that will never lose, so users can try their skill against the computer.

Programming a Tic Tac Toe game is not difficult and is generally considered to be a high school level exercise. My objective here is to learn the JavaScript language and to try out some best practices with respect to JavaScript design.
